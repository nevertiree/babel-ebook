**BabelEbook** is an EPUB translator powered by large language models.
It produces bilingual e-books where each translated paragraph is followed by
 the original text.

**BabelEbook（巴别塔）** 是一款基于大语言模型的 EPUB 翻译工具，
能把外文电子书翻译成中英双语版本。
